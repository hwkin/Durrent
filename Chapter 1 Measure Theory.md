## Chapter 1 Measure Theory

### 1.1 Probability Spaces

1. A **probability space** is a triple $(\Omega, \mathcal{F}, P)$, where $\Omega$ is a set of "outcomes", $\mathcal{F}$ is a set of "events", and $P:\mathcal{F}\rightarrow[0,1]$ is a function that assigns probabilities to events. We assume that $\mathcal{F}$ is a $\sigma$-field, which is closed under complement and countable union.

2. Without $P$, $(\Omega,\mathcal{F})$ is called a **measurable space**, and a measure is such a set function $\mu:\mathcal{F}\rightarrow\mathbf{R}$:

   (i) $\mu(A)\geq \mu(\emptyset) = 0\ \forall, A\in\mathcal{F}$ 

   (ii) if $A_i\in\mathcal{F}$ is a countable sequence of disjoint sets, then $\mu(\cup_i A_i)=\sum_i\mu(A_i)$

   if $\mu(\Omega)=1$, $\mu$ is called a probability measure, usually denoted by $P$.

   **Theorem 1.1.1** Let $\mu$ be a measure on $(\Omega,\mathcal{F})$

   (i) **monotonicity** if $A\subset B$ then $\mu(A)\leq\mu(B)$

   (ii) **subadditivity** if $A\subset\cup_{m=1}^\infty A_m$ then $\mu(A)\leq \sum_{m=1}^\infty \mu(A_m)$ 

   (iii) **continuity from below** if $A_i\uparrow A$ , then $\mu(A_i)\uparrow \mu(A)$

   (iv) **continuity from above** if $A_i\downarrow A$, then $\mu(A_i)\downarrow \mu(A)$ 

   **Example 1.1.1 Discrete probability spaces** Let $\Omega$ be a countable set and $\mathcal{F}$ be the set of all subsets of $\Omega$. Let
   $$
   P(A)=\sum_{\omega\in A} p(\omega),\ where\ p(\omega)\geq 0\ and\ \sum_{\omega\in\Omega}p(\omega)=1
   $$

(ii) if we are given a set $\Omega$ and a collection $\mathcal A$ of subsets of $\Omega$, the there is a smallest $\sigma$-field containing $\mathcal{A}$, which is called the $\sigma$-field generated by $\mathcal{A}$ and denoted by $\sigma(\mathcal{A})$.

3. Consider $\mathbf{R}^d$ and Borel sets $\mathcal{R}^d$ (the smallest $\sigma$-field containing the open sets) of it.

   **Example 1.1.2 Measures on the real line.** Measure on $(\mathbf{R},\mathcal{R})$ are defined by giving probability a **Stieltjes measure function** with the following properties:

   (i) $F$ is nondecreasing

   (ii) $F$ is right continuous.

   **Theorem 1.1.2** Associated with each Stieltjes measure function F there is a unique measure $\mu$ on $(\mathbf{R},\mathcal{R})$ with $\mu((a,b])=F(b)-F(a)$

   Especially, when $F(x)=x$, the resulting measure is called **Lebesgue measure**.

   Notice if $b_n\downarrow b$ then $(a,b_n]\downarrow (a,b]$, which correspond to right-continuity.

   Our goal is to prove this. The idea is to give measure on semialgebra, then build algebra,then build $\sigma$-algebra. 

   ***Definition*** 

   (i) A collection $\mathcal{S}$ of sets is called a **semialgebra** if it's closed under intersection and the completion of a set is a finite disjoint union of sets in $\mathcal{S}$. 

   (ii) A collection $\mathcal{S}$ of sets is called a **algebra** if it's closed under completion and union. 

   **Example 1.1.3.** $\mathcal{S}_d=\{\emptyset\}\cup\{(a_1,b_1]\times\cdots\times(a_d,b_d]\subset \mathbf{R}^d: -\infty\leq a_i<b_i\leq \infty\}$ is a semialgebra. when $d=1$, **Theorem 1.1.2** gives the measure on it. Notice the left-open property assures this.

   **Example 1.1.4** A $\sigma$-algebra is an algebra, but the converse is false. Let $\Omega=\mathbf{Z}$ and $\mathcal{A}=\{ A\subset \mathbf{Z} : A\ or\ A^c\ is\ finite \}$ is an algebra. Notice it's not a $\sigma$-algebra.

   Our goal is to build $\sigma$-algebra from semialgebra. 

   **Lemma 1.1.3** If $\mathcal{S}$ is a semialgebra then $\bar{\mathcal{S}}$ which consists of finite disjoint union of sets in $\mathcal{S}$ is an algebra, called the **algebra generated by** $\mathcal{S}$. This is direct from definition.

   **Example 1.1.5** Let $\Omega=\mathbf{R}$ and $\mathcal{S}=\mathcal{S}_1$ then $\bar{\mathcal{S}_1}=\{\emptyset\}\cup\{\cup_{i=1}^k(a_i,b_i]: -\infty\leq a_i <b_i\leq \infty\}$. 

   Given a set function $\mu$ on $\mathcal{S}$ , it can be extended to $\bar{\mathcal{S}}$ by $\mu(+_{i=1}^n A_i) = \sum_{i=1}^n \mu(A_i)$. 

   ***Definition*** 

   A **measure on an algebra** $\mathcal{A}$ is a set function $\mu$ with 

   (i) $\mu(A)\geq \mu(\emptyset)=0\ \forall A\in\mathcal{A}$ 

   (ii) if $A_i\in\mathcal{A}$ are disjoint and their union is in $\mathcal{A}$, $\mu(\cup_i A_i)=\sum_i\mu(A_i)$. 

   Notice this correspond with the definition of measure on a $\sigma$-algebra, while countable union may be not in the algebra. 

   $\mu$ is said to be $\sigma$-**finite** if there are countable sets in $\mathcal{A}$ , each of which is of finite measure with $\mu$, and their union is $\Omega$.

   **Theorem 1.1.4** Let $\mathcal{S}$ be a semialgebra and let $\mu$ defined on $\mathcal{S}$ satisfies $\mu(\emptyset)=0$. Suppose

   (i) if $S\in\mathcal{S}$, and $S=+_{i=1}^n S_i$ , where $S_i\in\mathcal{S}$, then $\mu(S)=\sum_i\mu(S_i)$ (finite additivity)

### 1.2 Distributions

1. a real valued function $X$ defined on $\Omega$ is a **random variable** if for every Borel set $B\subset\mathbf{R}$ we have $X^{-1}(B)=\{\omega: X(\omega)\in B\}\in\mathcal{F}$. When emphasizing the $\sigma$-field, we'll say that $X$ is $\mathcal{F}$-measurable or write $X\in\mathcal{F}$.

   **Example** 

   (i) If $\Omega$ is a discrete probability space, then any function $X:\Omega\rightarrow\mathbf{R}$ is a random variable. 

   (ii) **indicator function** of $A\in\mathcal{F}$: $1_A(\omega)=1,\ \omega\in A; 0,\ \omega\notin A$ 

   If X is a random variable, then $X$ induces a probability measure on $\mathbf{R}$ called its **distribution** by setting $\mu(A)=P(X\in A)$ for Borel set $A$. This pull $A\in \mathcal{R}$ back to $X^{-1}(A)\in\mathcal{F}$ and take P of that set. It's easy to verify this is a probability measure.

   The distribution of $X$ is usually described by its **distribution function**, $F(x)=P(X\leq x)$.

   **Theorem 1.2.1** Any distribution function $F$ has the following properties:

   (i) $F$ is nondecreasing

   (ii) $\lim_{x\rightarrow\infty} F(x)=1, \lim _{x\rightarrow-\infty} F(x)=0$

   (iii) $F$ is right continuous

   (iv)$F(x-):=\lim_{y\uparrow x}F(y)=P(X<x)$

   (v) $P(X=x)=F(x)-F(x-)$

   **Theorem 1.2.2** If $F$ satisfies (i),(ii),(iii) in **Theorem 1.2.1**, then it's the distribution function of some random variable.

   <font color='red'>_proof of Theorem 1.2.2_</font> 

   Let $\Omega=(0,1)$, $\mathcal{F}$ be the Borel sets, and $P$ be the  Lebesgue measure. If $\omega\in(0,1)$, let $X(\omega)=\sup\{y:F(y)<\omega\}$.

   Verify $\{\omega:X(\omega)\leq x\}=\{\omega: \omega\leq F(x)\}$: 

   If $X(\omega)\leq x$, $\sup\{y:F(y)<\omega\}\leq x$, since $F$ is nondecreasing, $x\notin \{y:F(y)<\omega\}$, so $F(x)\geq \omega$. 

   Conversely, if $\omega\leq F(x)$, then $X(\omega)=\sup\{y:F(y)<\omega\}\leq \sup\{y:F(y)<F(x)\}=x$.

   take $P$, $LHS=P(X\leq x)$, $RHS= F(x)$.

   Even though $F$ is not 1-1 or onto, we call $X$ the **inverse** of $F$ and denote it by $F^{-1}$. From the proof, we can conclude directly that for uniform distribution $U$, $F^{-1}(U)$ has the distribution function $F$. 

   ***Definition*** if $X$ and $Y$ induce the same distribution $\mu$ on $(\mathbf{R},\mathcal{R})$, we say they are **equal in distribution**, denoted by $X=_d Y$. They are equal in distribution if and only if they have the same distribution function.

2. ***Definition*** if $F(x)=\int_{-\infty}^x f(y)dy$ for some $f$, we say that $X$ has the density function $f$, denoted by $f_X$. $f$ is a density function if and only if $f\geq 0$ and $\int f=1$.

   **Example 1.2.1 Uniform distribution on (0,1)**  $f(x)=1, x\in(0,1);0,\ otherwise.$ $F(x)=0,x\leq 0;x,0\leq x\leq 1; 1,x\geq 1$.   

   **Example 1.2.2 Exponential distribution with rate** $\lambda$  $f(x)=\lambda e^{-\lambda x}, x\geq 0;0,\ otherwise$. $F(x)=0,x\leq 0; 1-e^{-x}, x\geq 0$

   **Example 1.2.3 Standard normal distribution** $f(x)=(2\pi)^{-1/2} \exp(-x^2/2)$ 

   **Theorem 1.2.3** For $x>0$,
   $$
   (x^{-1}-x^{-3})\exp(-x^2/2)\leq \int_x^\infty\exp(-y^2/2)dy\leq x^{-1}\exp(-x^2/2)
   $$
   ***Definition*** A distribution function on $\mathbf{R}$ is said to be **absolutely continuous** if it has a density and **singular** if the corresponding measure is singular w.r.t. Lebesgue measure. (Radon-Nikodym Theorem)

   **Example 1.2.4 Uniform distribution on the Cantor set** Let the distribution function be Cantor function. It doesn't have a density function, while under the measure it induces the completion of Cantor set is 0. So it's singular w.r.t. Lebesgue measure.

   ***Definition*** A probability measure $P$ (Or its corresponding distribution function) is said to be discrete if there is a countable set $S$ with $P(S^c)=0$.

   **Example 1.2.5 Point mass at 0** $F(x)=1$ for $x\geq 0$, $F(x)=0$ for $x<0$.

   **Example 1.2.6 Dense discontinuities** Let $q_1,q_2,...$ be an enumeration of rationals and Let $\alpha_i>0,\sum_{i=1}^\infty \alpha_i = 1$. Let $F(x)=\sum_{i=1}^\infty \alpha_i 1_{[q_i,+\infty)}$. Note: A distribution function has at most countable discontinuities since it's monotonic.

   **Exercise 1.2.5** Suppose $X$ has density $f$ and $\alpha\leq X\leq \beta\ a.s.$, $g$ defined on $(\alpha,\beta)$ is strictly increasing and differentiable. Then $g(X)$ has density $f(g^{-1}(y))/g'(g^{-1}(y))$ for $y\in(g(\alpha),g(\beta))$ and 0 otherwise. This formular is very useful. ### 1.4 Integration

Let $\mu$ be a $\sigma$-finite measure on $(\Omega,\mathcal{F})$. We define the integration of measurable functions by four steps.

_Step 1_ Simple functions

***Definition*** $\phi$ is said to be a simple function if $\varphi(\omega)=\sum_{i=1}^n a_i1_{A_i}$ and $A_i$ are disjoint sets with finite measure. 

The integral of $\varphi$ : $\int \varphi d\mu=\sum_{i=1}^n a_i\mu(A_i)$. It's easy to verify this is well-defined.

To ensure the integral we defined has general property of integrals, we need to claim this lemma repeatedly.

**Lemma 1.4.1** Let $\varphi$ and $\psi$ be simple functions, 

(i) If $\varphi>0$ a.e. then $\int\varphi\ d\mu\geq 0$

(ii) $\forall\ a\in\mathbf{R}$, $\int a \varphi\ d\mu=a\int\varphi\ d\mu$

(iii) $\int \varphi+\psi\ d\mu = \int\varphi d\mu+\int \psi d\mu$ 

**Lemma 1.4.2** If (i) and (iii) in **Lemma 1.4.1** hold, then:

(iv) if $\varphi\leq\psi$ a.e. then $\int \varphi d\mu\leq\int\psi d\mu$

(v) If $\varphi=\psi$ a.e. then $\int \varphi d\mu=\int\psi d\mu$

(vi) $|\int \varphi d\mu|\leq \int |\varphi| d\mu$

_Step 2_ Bounded functions

Let $E$ be a set of finite measure and $f$ be a bounded function that vanishes on $E^c$.

The integral of $f$: $\int f\ d\mu=\sup_{\varphi\leq f}\int\varphi d\mu=\inf_{\psi\geq f}\int \psi\ d\mu$ 

To verify that the integral are well-defined, notice (i) two limits are both finite (ii) they are equal.

The lemmas are also valid here.

_Step 3_ nonnegative functions

define the integral by $\int f\ d\mu=\sup\{\int h\ d\mu: 0\leq h\leq f, h\ is\ bounded\ on\ finite\ support\ set.\}$ 

**Lemma 1.4.4** Let $E_n\uparrow\Omega$ with $\mu(E_n)<\infty$, then $\int_{E_n}f\wedge n\ d\mu\uparrow\int f\ d\mu$.

By this, above lemmas can be verified safely.

_Step 4_ general function

We say $f$ is **integrable if $\int|f|d\mu<\infty$.

Let $f^+(x)=f(x)\vee 0$ and $f^-(x)=f(x)\wedge 0$, then they are measurable and $f=f^+-f^-$,$|f|=f^++f^-$

integral is defined by $\int f\ d\mu=\int f^+\ d\mu-\int f^-\ d\mu$

**Lemma 1.4.6** If $f=f_1-f_2$ where $f_i$ are nonnegative and integrable, then $\int f\ d\mu=\int f_1\ d\mu-\int f_2\ d\mu$.

<font color='red'>_proof of Lemma 1.4.6_</font> divide nto positive and negative parts and verify.

By this, above lemmas can be verified.

### 1.5 Properties of the Integral

We will develop some properties of the integration defined above.

**Theorem 1.5.1 Jensen's Inequality** Suppose $\varphi$ is **convex**, i.e. $\lambda\varphi(x)+(1-\lambda)\varphi(y)\geq \varphi(\lambda x+(1-\lambda)y)\ \forall\ \lambda\in(0,1), x,y\in\mathbf{R}$. 

If $\mu$ is a probability measure, $f$ and $\varphi(f)$ are integrable, then $\varphi(\int f\ d\mu)\leq \int\varphi(f)\ d\mu$.

<font color='red'>_proof of Theorem 1.5.1_</font> The idea is to use the support line at $c=\int f\ d\mu$.

Since $\lim_{h\downarrow 0}\frac{\varphi(c)-\varphi(c-h)}{h}\leq\lim_{h\downarrow 0}\frac{\varphi(c+h)-\varphi(c)}{h}$ , take $a$ between two limit, set $\mathcal{l}(x)=a(x-c)+\varphi(c):=ax+b$, then it always lies under the function. So $\int\varphi(f)d\mu\geq\int(af+b)d\mu=\mathcal{l}(c)=c$.

***Definition*** In $L_p$ space the norm is defined by $\|f\|_p=(\int|f|^pd\mu)^{1/p}$.

**Theorem 1.5.2 Holder' inequality** If $p,q\in(1,\infty)$ with $1/p+1/q=1$ then $\int |fg|d\mu\leq \|f\|_p\|g\|_q$

<font color='red'>_proof of Theorem 1.5.2_</font> If any of $\|f\|_p,\|g\|_q$ is 0, then $|fg|=0$ a.e. WLOG suppose $\|f\|_p=\|g\|_q=1$, use Young's inequality: $x^p/p+y^q/q\geq xy$, and integrate both side.

Now we are expecting the integration behaves well under taking limits.

***Definition***: $f_n\rightarrow f$ **in measure** if $\forall\varepsilon$, $\mu(|f_n-f|>\varepsilon)\rightarrow 0$.

**Theorem 1.5.3 Bounded Convergence theorem** Let $E$ be a set with finite measure and suppose $f_n$ vanishes on $E^c$, $|f_n(x)|\leq M$, and $f_n\rightarrow f$ in measure, then $\int f_n\ d\mu\rightarrow \int f\ d\mu$.

<font color='red'>_proof of Theorem 1.5.3_</font> $\forall\varepsilon$, Let $G_n=\{|f_n-f|<\varepsilon\}$ and $B_n=E-G_n$, then 
$$
|\int fd\mu-\int f_nd\mu|\leq\int|f-f_n| d\mu=\int_{G_n} |f-f_n|d\mu+\int_{B_n}|f-f_n|d\mu\leq \varepsilon\mu(E)+2M\mu(B_n)\rightarrow \varepsilon\mu(E)
$$
Since $\varepsilon$ is arbitrary, the proof is complete.

### 1.6 Expected Value

***Definition*** if $X\geq 0$ is a random variable on $(\Omega,\mathcal F,P)$, its **expected value** $EX=\int XdP$, which always make sense but may be $\infty$.For general case, $EX$ **exists** if $EX^+,EX^- <\infty$, and set $EX=EX^+-EX^-$. $EX$ is often called **mean**.

**Theorem 1.6.1** Suppose $X,Y\geq 0$ or they are integrable.

(a) $E(X+Y)=EX+EY$

(b) $E(aX+b)=aEX+b$

(c) $X\geq Y\Rightarrow EX\geq XY$

### 1.6.1 Inequalities

**Theorem 1.6.2 Jensen's Inequality** If $\varphi$ is convex, then $E\varphi(X)\geq\varphi(EX)$ if they make sense.

**Theorem 1.6.3 Holder's Inequality** If $p,q\in [1,\infty]$ and $1/p+1/q=1$, then $E|XY|\leq \|X\|_p\|Y\|_q$

**Notation** $E(X;A)=\int_A X dP$.

**Theorem 1.6.4 Chebyshev's inequality** Suppose $\varphi:\mathbf R\rightarrow\mathbf R$ is nonnegative, let $A\in\mathcal R$ and let $i_A=\inf\{\varphi(y):y\in A\}$,then
$$
i_AP(X\in A)\leq E(\varphi(X);X\in A)\leq E\varphi(X)
$$
<font color='red'>_proof of Theorem 1.6.4_</font> $i_A1_{X\in A}\leq \varphi(X)1_{X\in A}\leq \varphi(X)$ , then integrate.

_Remark_ A special case (very useful): $a^2P(|X|\geq a)\leq EX^2$

### 1.6.2 Integration to limit

**Theorem 1.6.5 Fatou' s lemma** If $X_n\geq 0$ then $\liminf_{n\rightarrow\infty}EX_n\geq E(\liminf X_n)$

**Theorem 1.6.6 Monotone convergence theorem** If $0\leq X_n\uparrow X$ then $EX_n\uparrow EX$

**Theorem 1.6.7 Dominated convergence theorem** If $X_n\rightarrow X$ a.s. and $|X_n|\leq Y$ and Y is integrable, $EX_n\rightarrow EX$.

**Theorem 1.6.8** Suppose $X_n\rightarrow X$ a.s.. Let $g,h$ be functions with 

(i)$g\geq 0$ and $g(x)\rightarrow\infty$ as $|x|\rightarrow\infty$ 

(ii) $|h(x)|/g(x)\rightarrow 0 $ as $|x|\rightarrow \infty$ 

(iii) $Eg(X_n)\leq K\leq\infty\ \forall n$. 

Then $Eh(X_n)\rightarrow Eh(X)$.

<font color='red'>_proof of Theorem 1.6.8_</font> The main idea is to bound $X_n$ and estimate the body and tail.

(i) bound X: Take M sufficient large such that $P(|X|=M)=0$, i.e. $|X|$ is continuous at $M$ and $g(x)>0$ when $|x|\geq M$. Denote $Y1_{|Y|\leq M}$ by $\tilde Y$ for a random variable $Y$.

(ii) estimate the body: since $\tilde X_n\rightarrow \tilde X$ and $h$ is continuous, $h(\tilde X_n)$ is bounded, so $Eh(\tilde X_n)\rightarrow Eh(\tilde X)$.

(iii) estimate the tail: $|Eh(\tilde Y)-Eh(Y)|\leq E|h(\tilde Y)-h(Y)|\leq E(|h(Y)|;|Y|>M)\leq \epsilon_M Eg(Y)$ for all random variable $Y$, where $\epsilon_M=\sup\{|h(x)|/g(x): |x|\geq M\}$.

(iii) total estimation: $|Eh(X_n)-Eh(X)|\leq |Eh(X_n)-Eh(\tilde X_n)|+|Eh(\tilde X_n)-Eh(\tilde X)|+|Eh(\tilde X)-Eh(X)|$. Taking limits, $\limsup RHS \leq 2\epsilon_M K$. Since $\epsilon_M\rightarrow 0$ as $M\rightarrow\infty$, the convergence hold.
